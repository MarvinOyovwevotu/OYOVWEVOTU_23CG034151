About This App

This web application uses a machine learning model to detect human emotions from images. Supported emotions: Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise.

Make sure your uploaded image is facing forward and not obstructed by sunglasses, masks, or extreme angles.
